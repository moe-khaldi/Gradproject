#JUSTutor

JUSTutor is an AI-based educational system, used by IT students to upload and chat with their subject material, generate quizzes to test themselves and get detailed and customizable explainations by a purely AI powered tutor. It can be supervised by an instructor to make adjustments to the AI respone or to view students' progress. JUSTutor is a web based system, that allows users to create accounts and save thier credintials for easier access and session management.
